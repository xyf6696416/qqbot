# 意图路由关键词表（拼接提示词之前）

## 匹配优先级（从高到低）

```
1. 显式存图/发图命令 → 覆盖其他
2. 含图片（用户自己发的图）→ vision
3. 引用图片/转发 → 结合文本
4. 关键词匹配
5. 模糊 → chat 兜底
6. 其他 → chat
```

## 1. send_img（发图）

### 显式命令
```
发图
发张
发两张|发三张|发几张|发N张
发一张
来张|来两张
来点|来点图
发色图
来个
发个图
发图片
```

### 带子文件夹
```
发萝莉|发萝莉图|萝莉图|来张萝莉
发脚|发脚图|脚图|来张脚
发大雷|大雷图
发轮奸|轮奸图
发同人|同人图
发色彩|色彩图
```

### 谐音/口语
```
整点图|整几张|整一个
搞张图|搞点图
给张图|给点图
来点好康|来点好康的
```

### 回复引用转发（"发出来看看"）
```
发出来|发到群里
发出来看看|发出来瞅瞅
看看这些图
```

### 参数提取规则
```
"两张" → count=2
"三张" → count=3
"几张" → count=1（视为1）
"一张/张" → count=1
无数量 → count=1
"萝莉" → src=萝莉
"脚" → src=脚
"大雷" → src=大雷
"同人" → src=同人
"色彩" → src=色彩图
"轮奸" → src=轮奸
无子文件夹 → src=null（从根目录随机选）
```

## 2. save_img（存图）

### 显式命令
```
存图|存张图
保存
存起来
存一下|存这张
收图|收藏
收了
```

### 带目标文件夹
```
存到萝莉|保存到萝莉
存到脚|保存到脚
存到大雷
存到同人
```

### 回复引用
```
存了|存好啦|存好了    ← 引用图片时
这张存|这张留着
```

### 引用转发批量存
```
全存|全部存|全保存
存这些|都存了
里面的图都存
```

### 单张存（转发中第几张）
```
第1张存|第2张保存
第一张存|第二张保存
```
（配合转发缓存定位）

## 3. vision（识图）

### 显式识图
```
这是啥|这是什么
这啥
图里有什么|图片里有什么
看看|让我看看
识别|识别这张|识别这个
帮我看看|帮我看
帮我认一下
```

### 问图内容
```
这是什么图
这上面是谁|这是谁
这什么动物|啥动物
这是什么花|什么品种
这写的什么|这上面写的啥
```

### 引用转发识图
```
看看这张|看看这个
第N张看看|第3张是什么
里面的图是啥
```

### 自动触发（不需要关键词）
```
has_image = true AND 消息非纯"发图/存图"命令
引用图片 = true AND 消息非纯"发图/存图"命令
```

## 4. chat（正常对话）

**不匹配以上任何关键词 + 不含图片 = chat**

例外（含发/存字但不是命令）：
```
"我去发个消息给老板"  → chat
"这个图发给我看看"    → chat（这里"发"是分享的意思）
"你保存一下聊天记录"  → chat
"我存了个链接"       → chat
```

这种模糊 case 走 chat 兜底。

## 5. 优先级裁定规则

### 含图片附件（has_image=true）
```
文本中含 "存"/"保存"/"存到"  → save_img（存用户自己发的图）
文本中含 "发"/"来张"/"色图" → send_img（互相矛盾，但不常见）
文本中含 vision 关键词      → vision
其他                      → vision（默认用户发图就是想问）
```

### 引用图片（has_ref_image=true）
```
文本中含 "存"/"保存"      → save_img
文本中含 "发"/"发出来"    → send_img
文本中含 vision 关键词    → vision
其他                    → chat（用户只是评论一下）
```

### 引用转发（has_ref_forward=true）
```
文本中含 "存"/"保存"      → save_img（批量存转发中图片）
文本中含 "第N张存"       → save_img（定位存）
文本中含 "发"/"发出来"    → send_img（从缓存重发）
文本中含 "第N张看看"     → vision（定位看）
文本中含 vision 关键词   → vision
其他                    → chat
```

## 6. 关键词匹配代码实现示例

```python
import re

# ===== 关键词模式（可配置） =====
SEND_IMG_PATTERNS = [
    r"发图|发张|发几张?|发一?张|来张|来点|来一?张|发色图|来个",
    r"发萝莉|发脚|发大雷|发轮奸|发同人",
    r"整点图|整?几张?|搞张图|给张图",
    r"来点好康",
    r"发出来|发到群里",
]

SAVE_IMG_PATTERNS = [
    r"存图|保存|存起来|存一下|存这张|收图|收藏|收了",
    r"存到\w+|保存到\w+",
    r"全存|全部存|都存了",
    r"第\w+张存|第\w+张保存",
]

VISION_PATTERNS = [
    r"这是啥|这是什么|这啥",
    r"图里|图片里",
    r"看看|让我看看|瞅瞅",
    r"识别|帮我看看|帮我看|帮我认",
    r"第\w+张看看",
    r"什么图|谁啊|是谁",
]

def extract_count(text):
    """从文本中提取发图数量"""
    m = re.search(r"(\d+)张", text)
    if m:
        return min(int(m.group(1)), 6)  # 最多6张
    return 1

def extract_src(text):
    """从文本中提取子文件夹名"""
    src_map = {
        "萝莉": ["萝莉", "loli"],
        "脚": ["脚", "足", "脚丫"],
        "大雷": ["大雷"],
        "轮奸": ["轮奸", "轮"],
        "同人": ["同人"],
        "色彩图": ["色彩", "色图"],
    }
    for folder, keywords in src_map.items():
        for kw in keywords:
            if kw in text:
                return folder
    return None

def extract_save_to(text):
    """从文本中提取存图目标文件夹"""
    m = re.search(r"(?:存到|保存到)\s*(\w+)", text)
    if m:
        return m.group(1)
    return None

def extract_nth(text):
    """从文本中提取第N张"""
    m = re.search(r"第(\d+)张", text)
    if m:
        return int(m.group(1))
    return None

def keyword_route(text, has_image=False, has_ref_image=False, has_ref_forward=False):
    """关键词匹配路由"""
    text_lower = text.lower()
    
    # 优先检测存图（权限相关）
    if any(re.search(p, text_lower) for p in SAVE_IMG_PATTERNS):
        return {
            "intent": "save_img",
            "confidence": "high",
            "params": {
                "save_to": extract_save_to(text),
                "target_index": extract_nth(text),
            }
        }
    
    # 发图
    if any(re.search(p, text_lower) for p in SEND_IMG_PATTERNS):
        return {
            "intent": "send_img",
            "confidence": "high",
            "params": {
                "count": extract_count(text),
                "src": extract_src(text),
            }
        }
    
    # 含图片 → vision
    if has_image or has_ref_image:
        return {
            "intent": "vision",
            "confidence": "high",
            "params": {}
        }
    
    # 识图关键词
    if any(re.search(p, text_lower) for p in VISION_PATTERNS):
        return {
            "intent": "vision",
            "confidence": "medium",
            "params": {
                "target_index": extract_nth(text),
            }
        }
    
    # 含引用转发但无明确关键词 → 模糊（走 SF）
    if has_ref_forward:
        return {
            "intent": "chat",
            "confidence": "low",  # 走 chat 兜底
            "params": {}
        }
    
    # 其他 → chat
    return {
        "intent": "chat",
        "confidence": "high",
        "params": {}
    }
```
